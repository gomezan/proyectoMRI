En esta carpeta se presentan 3 carpetas importantes:

*DCSRNomagdy: Este contiene la versión del DCSRN de Omagdy con posibles cambios hechos por nosotros. El repositorio original se puede encontrar:
https://github.com/omagdy/DCSRN
*DCSRNyunzeman: Este contiene la versión del DCSRN de Yunzeman con posibles cambios hechos por nosotros. El repositorio original se puede encontrar:
https://github.com/YunzeMan/DCSRN
*Admin Conjunto de datos: Carpeta con los archivos necesarios para crear o modificar el conjunto de datos. Dentro de esta carpeta se encuentra un Readme con mayor información. 

La ejecución de estos scripts puede realizarse por consola o utilizando el main.py quien puede ejecutar cualquier script utilizando la ubicación de este.
